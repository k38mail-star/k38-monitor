## 五、GitHub 有价值功能调研

### 高价值（可直接借鉴）
1. **Glances** (github.com/nicolargo/glances) — Python 系统监控，有 Web UI、REST API、告警
   - 核心亮点: 传感器检测、容器监控、进程列表
   - 可借鉴: 统一 JSON API 设计、告警触发器

2. **netdata** (github.com/netdata/netdata) — 单机最强，但大而全
   - 核心亮点: 每指标 sparkline + 菜单树 + 1s 间隔
   - 可借鉴: 热力图标配色方案、卡片展开/折叠

3. **s-tui** (github.com/amanusk/s-tui) — 终端 GPU 监控
   - 核心亮点: GPU 功耗曲线 + 显存占用的直观展示
   - 可借鉴: 终端交互模式（watch 命令扩展）

### 中期价值
4. **uptime-kuma / statping-ng** — 服务可用性监控 + 告警推送
5. **dockprom** — Docker + Prometheus + Grafana 组合

### 不建议做
- ❌ Prometheus + Grafana 太重，违背零依赖原则
- ❌ Nagios/Zabbix 太老
